 # GIT Desarrollo Colaborativo



 Esta herramienta funciona como un sistema de control de versiones e implementa un

 \_\_grafo dirigido acíclico\_\_ para llevar el control de los cambios a lo largo de la historia del proyecto.



&nbsp;Este grafo se forma mediante la creación de nodos (vértices) que apuntan a otros realizados con anterioridad \_(aristas)\_, para identificar cuáles fueron las acciones realizadas históricamente
